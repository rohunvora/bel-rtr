# Belief Router

A Perplexity Finance-style interface for expressing market beliefs and trading.

## Features

- **Natural language input** - Type your market belief in plain English
- **Trade suggestions** - Get structured trade recommendations with leverage, timeframe, and risk levels
- **Long/Short actions** - Simple one-click execution buttons
- **Live position tracking** - Real-time PnL and time remaining
- **Fast Play mode** - Get instant momentum-based suggestions

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion
- Lucide React Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Demo Beliefs

Try these example inputs:
- "I think SOL will pump tomorrow"
- "MSTR is going to moon"
- "ETH looks weak here"
- "BTC to 100k this week"

## Note

This is a **frontend-only prototype** for UX validation. No real trading functionality is implemented.
